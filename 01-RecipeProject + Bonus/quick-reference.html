HTML & CSS Quick Reference Guide

HTML Tags

The following are some common HTML tags with examples:

  • <h1>: Main heading
  • <p>: Paragraph text
  • <ul>: Unordered list
  • <ol>: Ordered list
  • <li>: List item
  • <div>: Section or container
  • <span>: Inline container
  • <a>: Anchor (link)
  • <img>: Image
  • <code>: Inline code
  • <pre>: Preformatted text

Code Examples

Below is an example of HTML code using various tags:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Example Page</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

CSS Properties

Here are some common CSS properties with examples:

  • color: Sets the text color
  • background-color: Sets the background color
  • font-size: Sets the size of the font
  • margin: Sets the outer spacing of an element
  • padding: Sets the inner spacing of an element
  • border: Sets the border around an element
  • display: Specifies the display behavior (block, inline, etc.)
  • text-align: Sets the horizontal alignment of text

CSS Code Example

Below is an example of CSS code:

body {
  background-color: #f4f4f4;
  color: #333;
  font-size: 16px;
  margin: 0;
  padding: 20px;
}

h1 {
  text-align: center;
  color: #4CAF50;
}

p {
  margin: 10px 0;
  line-height: 1.6;
}
